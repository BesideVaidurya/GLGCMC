# GLGCMC MATLAB Code

This folder contains the MATLAB implementation of **Globality and Locality-Guided Collaborative Learning for Anchor-Graph Multi-view Clustering (GLGCMC)**.

## Folder structure

```text
GLGCMC/
|-- demo.m          Main script for running experiments
|-- train.m         Main optimization function of GLGCMC
|-- utils/          Utility functions
|-- data/           Multi-view datasets
`-- result/         Experimental results generated by demo.m
```

## Data preparation

Please put the `.mat` dataset files into the `data/` folder. The current code searches dataset files by name and does not rely on a fixed absolute path.

Each dataset file should contain multi-view features and labels. The code supports the following variable names:

- Feature variable: `X` or `data`
- Label variable: `Y`, `y`, `gt`, or `truelabel`

The feature matrix of each view can be stored as either `n x d` or `d x n`; the script will automatically adjust it to the required format.

## How to run

1. Open MATLAB.
2. Set the current folder to this code directory:

   ```matlab
   cd('path_to_your_code/GLGCMC')
   ```

3. Open `demo.m`.
4. Select the dataset to run by editing `Data_list`, for example:

   ```matlab
   Data_list = {
       'MSRCv1.mat'
   };
   ```

5. Set the hyperparameters according to the table below:

   ```matlab
   lamda1_list = [10];
   lamda2_list = [10];
   m_multiplier_list = [4];
   ```

6. Run:

   ```matlab
   demo
   ```

7. The results will be saved in the `result/` folder. For example:

   ```text
   result_MSRCv1.mat
   result_MSRCv1.txt
   result_MSRCv1.xlsx
   result_MSRCv1_best.mat
   result_MSRCv1_best.txt
   result_MSRCv1_best.xlsx
   ```

## Optimal hyperparameter settings

Here, `k` denotes the number of clusters, and the anchor number is set as `m = r k`. In `demo.m`, this is implemented by setting:

```matlab
m_multiplier_list = [r];
```

| Dataset | File name | `lambda_1` | `lambda_2` | Anchor number |
|---|---|---:|---:|---:|
| MSRC_v1 | `MSRCv1.mat` | 10 | 10 | `m = 4k` |
| ORL | `ORL.mat` | 100 | 10 | `m = 3k` |
| Mfeat | `Mfeat.mat` | 10 | 100 | `m = 5k` |
| MNIST-4 | `MNIST-4.mat` | `1e3` | `1e4` | `m = 4k` |
| CCV | `CCV.mat` | 0.01 | `1e6` | `m = 3k` |
| Hdigit | `Hdigit.mat` | 1 | `1e3` | `m = 4k` |

## Notes

- `demo.m` automatically adds the `utils/` folder to the MATLAB path.
- The random seed is fixed by `rng('default')` for reproducibility.
- The script reports ACC, NMI, Purity, AR, Recall, Precision, F-score, standard deviations, runtime, and running status.
- If multiple files with the same dataset name are found, the script will use the first matched file and print the selected path.
